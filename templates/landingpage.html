Music in Context

Discover Music in Context

A project by John Tokarowski


This app currently offers 3 use case.

1) Subgenre analysis - all songs in selected user playlists will be analyzed using a clustering algorithm, resulting in 5 new playlists determined by musical similarity.

2) Playlist analysis - all songs in 1 selected user playlist will be pulled into a radar and line chart, to visually examine patterns in playlist construction

3) Beta Functionality: the model will retrieve the user's top recent tracks and build a pool of recommendations. The model will then fit the recommendations to a 'curve' and present the complete set as a playlist in Spotify

This tool will ask you for Spotify Authentication to create new playlists. No playlists will be deleted.
